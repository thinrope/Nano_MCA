Nano_MCA
========

Firmware to use DSO Nano as a Multi-Channel Analyzer (MCA) for gamma spectroscopy.

Initially developed by LeDoyle Pingel as GammaGrapherVersion2 (GGV2) with many beta-testers from GammaSpectrometry Yahoo group.


http://www.seeedstudio.com/wiki/DSO_Nano

http://sourceforge.net/projects/gammagrapherv2/

http://tech.groups.yahoo.com/group/GammaSpectrometry/

I will try to clean up some of the code and improve the build process.
